# WirdiApp 🌿

Application de suivi de révision du Quran — Progressive Web App.

## Déployer sur Vercel (5 minutes)

### Option A — Sans code (le plus simple)

1. Va sur [vercel.com](https://vercel.com) et crée un compte gratuit
2. Clique **"Add New Project"**
3. Choisis **"Browse"** et uploade le dossier `wirdiapp`
4. Vercel détecte Vite automatiquement → clique **Deploy**
5. Ton app est en ligne ! Tu reçois un lien type `wirdiapp.vercel.app`

### Option B — Via GitHub

1. Crée un repo GitHub et push ce dossier
2. Sur Vercel, importe le repo GitHub
3. Deploy automatique à chaque mise à jour

---

## Ajouter sur l'écran d'accueil iPhone

1. Ouvre Safari sur ton iPhone
2. Va sur ton lien Vercel
3. Tape l'icône **Partager** ↑ (en bas de Safari)
4. "Sur l'écran d'accueil"
5. "Ajouter"

L'app apparaît comme une vraie app, plein écran, sans barre Safari. ✓
